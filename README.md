# Pocket Tanks: Next-Gen Web Edition 🚀

A high-performance, retro-inspired artillery shooter built with vanilla JavaScript and HTML5 Canvas. This project reimagines the classic "Pocket Tanks" mechanics with modern web technologies, featuring physics-based terrain destruction, dynamic particle systems, and a variety of tactical weaponry.

## ✨ Key Features

* **Destructible Terrain:** Real-time terrain deformation using canvas manipulation and physics-based "settling" for tanks.
* **Diverse Arsenal:** 10+ unique weapons including:
    * **Tactical:** Shields and Bouncy Balls.
    * **Heavy Hitting:** "Big Daddy" and Napalm with lingering fire effects.
    * **Specialized:** Airstrikes and Cluster Bombs.
* **Advanced Physics:** Gravity, projectile trajectories, collision detection, and camera shake effects for immersive feedback.
* **Responsive UI:** A sleek, minimalist dark-themed interface inspired by modern design standards.
* **Audio Engine:** Integrated sound system for spatial feedback (firing, explosions, and UI interactions).

## 🛠 Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables & Blur filters).
* **Graphics:** Canvas API (2D Context).
* **Logic:** Vanilla JavaScript (ES6+ Classes and Modules).
* **Audio:** Web Audio implementation for SFX and background music.
